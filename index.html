Tilting Card Documentation

Getting Started

To start using the tilting card module, follow these simple steps.

Setting Up

1. Add the following lines to your HTML file to include the CSS and JS files: 

<link rel="stylesheet" href="https://github.com/sherpadndabambi/tilting-card/releases/download/v1.0.0/tilting-card.css"/>
<script src="https://github.com/sherpadndabambi/tilting-card/releases/download/v1.0.0/tilting-card.js"></script>

Usage

1. Add the tilting-card class to any container element in your HTML where you want the tilting card effect to appear. For example:

Usage Example

HTML CSS 
<head>

  <link rel="stylesheet" href="./assets/css/styles.css" />

  <link rel="stylesheet" href="https://github.com/sherpadndabambi/tilting-card/releases/download/1.0.0/tilting-card.css" />

  <script src="https://github.com/sherpadndabambi/tilting-card/releases/download/1.0.0/tilting-card.js"></script>

</head>

<body>

  <div class="card highlight-circle tilting-card">

    <h3>Tilting Card</h3>

    <p>
      JavaScript calculates the mouse position relative to the card's
      center and uses CSS transforms rotateX and rotateY to tilt the card.
    </p>

  </div>

</body>
.card {
  background: rgb(44,62,80);
  border-radius: 10px;
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
  color: #fff;
  padding: 20px;
  width: 250px;
}

Tilting Card

JavaScript calculates the mouse position relative to the card's center and uses CSS transforms rotateX and rotateY to tilt the card.
